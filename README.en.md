# Project realised during _Databases_ course
The topic of the project is a system called *athleet* used for organizing athletics competitions - in particular, for registering athletes for competitions with a limited number of places and organizing these competitions by the organizers 🎽.

# About the project
_Athleet_ is a backend application that allows you to create athletes, coaches, competitions, and athletic events. It allows you to register these athletes for competitions, add events to competitions, and cancel registrations if necessary. In addition, it gives you the ability to generate reports on registrations for competitions, athletes from a given club or coach, or events taking place within a given meeting 📝.

---
**Database management system:** MongoDB  
**Programming language:** Java  
**ORM framework:** Hibernate
